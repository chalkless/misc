# Jupyter Notebook
- プログラミングするときにターミナルとか開いて、つどつどコマンド打って、とやっていたのを、実行のかたまりとしてセル（あるいはブロック的な）ごとに実行したりドキュメントつけたりして、インタラクティブに試してみたりできるような環境
- Google ColaboratoryとかRStudioのR Markdownとかと似た感じ

## インストール
### condaの場合
```
conda create -n jupyter     # 一応、それ用の仮想環境を作る
conda install jupyter
```
### pipの場合
```
pip install jupyter
```
